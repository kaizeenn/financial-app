# Contributing

Terima kasih sudah berkontribusi 🙌

## Cara kontribusi
1. Fork repository ini
2. Buat branch baru: `feature/nama-fitur`
3. Commit perubahan yang jelas dan kecil
4. Buat Pull Request dengan deskripsi yang rapi

## Standar singkat
- Gunakan nama commit yang jelas
- Jangan kirim file build/dependency cache
- Pastikan perubahan bisa dijalankan
